== Description ==

By using WhatsApp Button Plugin, you can add a share button for WhatsApp into posts/pages.


Features:

* add a share button for WhatsApp into posts/pages
* Using start/end element
* You can allow/disallow button
* You can insert button in top/bottom content

Support: [EDC](http://www.edc.org.kw).

== Installation ==

* Upload the WhatsApp Button plugin to your blog, Activate it.
* Go to WhatsApp Button from menu.
